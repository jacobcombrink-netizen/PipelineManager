Built for creators who run structured, repeatable AI render pipelines and need to track it all without living in a spreadsheet.

Archetype & Character Library
Define Concept archetypes (original characters) and Meta archetypes (common tropes). Successful Metas seed new Concepts with built-in marketing vectors. Attach splash images with drag-drop or Ctrl+V paste.
üß©
Ingredient System
Categorise your animation building blocks ‚Äî actions, motions, emotions, styles. Assign short codes. Define compatibility rules to flag which ingredients conflict or must appear together.
üî®
Job Builder + Ideation Mixer
Build deliberate render jobs by selecting a character, output type, and specific ingredients. Or use the dashboard Ideation Mixer to roll random combinations with per-component lock/unlock controls.
üìù
Project Journal & Prompt Queue
Write prompts in the journal, allocate them to a project, and queue them for use. One click copies to clipboard and marks as collected. Flag prompts for revision, mark them done, or reset.
üé¨
Streamlined Media Import
Link a file to a render job and title, tags, character, and output type auto-fill from the job. The prompt used is saved alongside. Batch-fill SEO fields later when you have time.

Top Layer Composites
Link multiple render jobs to a composite clip. One click aggregates all their tags, descriptions, and SEO data ‚Äî ready to copy-paste for publishing.

Native Floating Dock
A tkinter always-on-top window that floats over your render software. Select a job, load its prompt queue, copy prompts to clipboard, and submit media with a file-browser dialog ‚Äî without touching the browser.

Pipeline Funnel Metrics
Dashboard tracks all possible combinations across your library versus how many are planned, rendered, imported, and fully tagged. Know exactly how much work is left at every stage.
